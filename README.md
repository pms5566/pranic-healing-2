# Pranic Healing Academy Single-Page Website

A high-fidelity, premium single-page demo website for the **Pranic Healing Academy** course portal.

## 🎨 Design System & Aesthetics
- **Theme Vibe**: Calm, spiritual, and professional.
- **Color Palette**:
  - Deep Purples (`#3b1f6e`): Representing spiritual depth, calm, and authority.
  - Soft Gold Accents (`#c9a84c`): Representing warmth, energy, and premium details.
  - Warm Off-White Background (`#faf7f2`): Gentle and comforting background.
- **Typography**:
  - Headings: *Cormorant Garamond* (Google Fonts) for elegant serif display.
  - Body Text: *Inter* (Google Fonts) for modern, high-legibility sans-serif reading.
- **Visuals**: 100% custom inline SVGs for all illustrations, course icons, section dividers, and status indicators.

## 🛠️ Technical Implementation
- **Zero Frameworks / Zero External Assets**: Fully contained in a single self-sufficient `index.html` file (with inline CSS and vanilla JS).
- **Responsive Layout**: Designed mobile-first, ensuring smooth column shifting, fluid scaling, and customized mobile drawer navigation.
- **Scroll Reveal**: Utilizing the browser's native `IntersectionObserver` to trigger fade-in animations as users scroll.
- **Dynamic Counters**: JavaScript-driven numeric counters that count up on initial page load.
- **Contact Form & Modal**: Built-in modal dialog trigger upon form submit, with backdrop click-to-close and Escape key listeners.
- **Floating CTA**: Fixed bottom-right pulsing WhatsApp bubble link.
